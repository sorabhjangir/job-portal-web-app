# Job Portal Web App

A full-stack job portal web application designed to connect job seekers with employers through an intuitive and responsive interface.

This project demonstrates real-world web development skills across the frontend and backend, with a focus on clean UI, structured code, and practical functionality.

---

##  Overview

The Job Portal Web App allows users to explore job listings, submit applications, and manage postings. It serves as both a **career discovery platform for applicants** and a **job posting system for employers**.

---

##  Features

-  **Job Browsing**  
  Users can view available job listings with relevant details.

-  **Job Application**  
  Job seekers can submit applications for positions.

-  **Employer Dashboard**  
  Employers can post, update, and manage job listings.

-  **User Authentication**  
  Secure signup and login system with role-based access (applicant/employer).

-  **Responsive UI**  
  Designed to work smoothly on desktop and mobile devices.
